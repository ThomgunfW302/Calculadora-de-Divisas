Calculadora de Divisas
Este es un proyecto sencillo de práctica donde se utiliza PHP, HTML y CSS para crear un interfaz funcional. Este es un programa que sirve para convertir el peso colombiano(COP) a 3 divisas diferentes:
- Doláres Estadounidenses (USD)
- Euro (EUR)
- Yen Japones (JPY)

INSTALACIÓn
Este proyecto funciona bajo un servidor local con XAMPP y Apache, se requiere que este prendido este servidor a la hora de abrir el programa y funcione correctamente.


MODO DE USO
Primero le va a pedir un monto en pesos colombianos, usted va a escribir la cantidad que desea convertir, luego en la casilla inferior, va a tener la opción de elegir la divisa, puede elegir una entre las 3 disponibles. Al darle al botón de convertir, aparecerá una pequeña ventana que se pone al frente junto con el resultado de la conversión y la moneda que eligió con la opción de volver y hacer otra.

!ADVERTENCIA! 
Este proyecto solo funciona con pesos colombianos ya que las tasas de conversión son fijas. Tambien puede que al día que lo use, pueda estar con un valor diferente ya que las tasas de conversión cambian cada día.
